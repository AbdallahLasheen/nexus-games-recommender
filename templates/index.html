NCF Architecture (Lecture 7)

  Input:  (user_id, item_id)
      ↓            ↓
  User Embedding  Item Embedding   (dim = 64)
      └────────────┘
           Concatenate  →  [128-dim vector]
               ↓
         FC(128 → 64) + ReLU + Dropout(0.2)
               ↓
         FC(64  → 32) + ReLU + Dropout(0.2)
               ↓
         FC(32  → 16) + ReLU
               ↓
         FC(16  →  1) + Sigmoid
               ↓
         Output: predicted preference ∈ (0, 1)
